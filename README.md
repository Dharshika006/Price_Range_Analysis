# Price Range Analysis

## Project Overview

This project analyzes restaurant price ranges and ratings using restaurant dataset features.

The analysis includes:

* most common price ranges
* average ratings by price range
* rating color analysis
* price distribution insights

---

# Features

* Price range frequency analysis
* Average rating calculation
* Rating color analysis
* Data visualization
* Restaurant pricing insights
